# CompostMarketplace
Compositor

App developed with Android Studio.
Backend script: php
Database: MySQL
Web Server: XAMPP and phpMyAdmin.
logs table contains sample latitude and longitude coordinates for the heatmap.

Compositor.zip: Contains the main application.
compositorScripts.zip: Contains all the backend scripts for the main application(in Compositor.zip).
databaseSchema.png: A schema diagram for the database.
MyApplication2.zip: Contains the heatmap section.
heatMapScripts.zip: Contains all the backend scripts for the heatmaps application(in MyApplication2.zip).
waste_management.sql: Exported sql file of the database. Can be imported to get the database.
